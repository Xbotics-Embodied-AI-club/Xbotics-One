# GRPO VLM CLEVR Data Manifests

This directory contains only lightweight manifests and classroom examples.

Large public data stays in the Hugging Face cache selected by `HF_HOME`:

```text
${HF_HOME}
```

Training uses `leonardPKU/clevr_cogen_a_train`. Evaluation uses a fixed 200-item SuperCLEVR counting subset recorded in `eval_superclevr_200.json`.
